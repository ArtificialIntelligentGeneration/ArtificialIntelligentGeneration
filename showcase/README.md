# Showcase projects

This directory contains sanitized public excerpts from real local systems. The originals include private runtime files, sessions, logs, customer materials, and local state, so the public versions keep the architecture and core code while removing secrets and personal data.

## Contents

- [`aisana-agent-os`](./aisana-agent-os) - Personal Agent OS / Infinity-style local agent workspace: runner, policy checks, tool gateway, memory, and approval flow excerpts.
- [`telegram-job-hunter`](./telegram-job-hunter) - Telegram-driven job lead parser/router with scoring, dedupe, notifications, and operator review.
- [`telegram-finance-bot`](./telegram-finance-bot) - Telegram expense importer and finance check-in workflow for personal cashflow tracking.
- [`ai-outbound-agent`](./ai-outbound-agent) - outbound workflow notes for lead discovery, website analysis, personalized drafts, and inbox monitoring.
- [`maker-framework`](./maker-framework) - MAKER-style decomposition, first-to-ahead-by-k voting, red-flag checks, and k-min estimation for long-running LLM tasks.

Safety note: no `.env`, tokens, `.session` files, chat exports, databases, client CSVs, logs, or NDA-bound code are included.
